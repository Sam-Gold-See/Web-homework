Web前端笔记

HTML

HTML 指的是超文本标记语言：HyperText Markup Language，属于一种标记语言，即是一套标记标签

HTML 用标记标签来描述网页

HTML 文档包含了 HTML 标签以及文本内容，也称作Web 页面

HTML基本结构

  • <!DOCTYPE html> 声明为 HTML5 文档
  • <html> 元素是 HTML 页面的根元素
  • <head> 元素包含了文档的元（meta）数据，如 <meta charset="utf-8"> 定义网页编码格式为 utf-8
  • <title> 元素描述了文档的标题
  • <body> 元素包含了可见的页面内容（仅在该范围内的内容可见）
  • <h1> 元素定义一个大标题
  • <p> 元素定义一个段落

在浏览器中键入F12按键开启调试模式

HTML标签

  • HTML 标签是由尖括号包围的关键词，比如 <html>
  • HTML 标签通常是成对出现的，比如 <b> 和 </b>
  • 标签对中的第一个标签是开始标签，第二个标签是结束标签

<标签>内容</标签>

HTML元素

HTML元素包括了开始标签和结束标签

  • HTML元素以开始标签起始，以结束标签终止
  • 元素的内容是开始标签和结束标签之间的内容
  • 某些HTML元素具有空内容
  • 空元素（没有内容的HTML元素）在开始标签中进行关闭
  • 大多HTML元素拥有属性
  • HTML元素之间可以嵌套

HTML属性

  • HTML 元素可以设置属性
  • 属性可以在元素中添加附加信息
  • 属性一般描述于开始标签
  • 属性总是以名称/值对的形式出现，比如：name="value"，属性值应该始终在引号内，如果属性值本身含有双引号，则必须使用单引号

HTML基础

  • <h1>-<h6>标签定义标题
  • <p>标签定义段落
  • <a href="URL">标签定义链接
  • <img src="URL" width="~" hight="~">标签定义图像
  • <hr>创建水平线
  • <!-- -->创建注释
  • <br\>作为换行符

HTML链接

<a>（锚）元素常用来创建链接

HTML链接元素属性

href：是链接最重要的属性，用于指定链接目标的URL可以是另一个网页的URL、文件的URL或者是其他资源的URL

target：指定链接如何在浏览器中打开 _blank为值时为在新标签页、窗口打开链接 _self为值时在当前标签页或窗口打开链接

id：使用id属性可用于创建一个HTML文档书签可以在href属性值中的URL的末端加上#(id)(无空格)跳转到指定位置，如果是对本链接则可以直接使用#(id)

HTML图像

HTML表格属性

border：<table>中的边框属性，赋值大小为边框外围的宽度

cellpadding：<table>中的属性，用于设置单元格边距（单元格大小）

cellspacing：<table>中的属性，用于设置单元格间距，默认为1

colspan：<th>、<td>中的属性，用于实现单元格跨列效果

rowspan：<th>、<td>中的属性，用于实现单元格跨行效果

HTML列表

HTML列表分为无序列表<ul>和有序列表<ol>，两种列表中都有由<li>定义的列表项

有序列表<ol>和无序列表<ul>之间可以相互嵌套

自定义列表使用<dl>作为自定义列表的标志，<dt>作为自定义列表的项，<dd>作为自定义列表中项的描述

HTML列表属性

type（<ol>）：有序列表序号随着赋值而改变A, a, I, i，依次为大写字母列表、小写字母列表、罗马数字列表、小写罗马数字列表

style="list-style-type:" (<ul>)：无序列表序号随着英文冒号后赋值而改变disc, circle, square，依次为实心圆列表、空心圆列表、正方形列表

HTML区块元素和内联元素

大多数HTML元素被定义为区块元素或内联元素

HTML区块元素

区块元素在浏览器显示的时候通常会以新行来开始或结束 如<h1>、<p>、<ul>、<table>

<div>是块级元素，可用于作为组合其他HTML元素的容器，该元素没有特定含义，如果和CSS一起使用，可以对大的内容块设置样式属性 其次可以用于文档布局

HTML内联元素

内联元素在显示时通常不会以新行开始 如<b>、<td>、<a>、<img>

<span>是内联元素，可用作文本的容器，也没有特定含义，在和CSS一同使用时，可以用于为部分文本设置样式属性

HTML布局

可以使用<div>进行HTML布局 使用属性float:left/right来控制区块的浮动属性

HTML表单和输入

HTML表单用于收集用户的输入信息 表单用于表示文档中的一个区域，此区域包涵交互控件，将信息发送到Web服务器

HTML表单包涵各种输入字段、复选框、单选按钮、下拉列表等元素

  • <form>元素用于创建表单，action属性定义了表单数据提交的目标URL，method属性定义了提交数据和HTTP方法

  • <label>元素用于为表单添加标签，提高文件可读性，便于区分表单内不同的输入内容

  • <input>元素用于创建文本输入框、密码框、单选按钮、复选框等。type属性定义了输入框的类型，id属性用于关联<label>元素，name属性用于标识表单字段

HTML表单

表单是一个包含表单元素的区域。 表单是允许问用户在表单输入内容，比如：文本域（textarea）、下拉列表（select）、单选框（radio-buuttons）、复选框（checkbox）等等

输入元素

HTML统一资源定位器（URL）

scheme://host.domain:port/path/filename

  • scheme - 定义因特网服务的类型。最常见的类型是 http
  • host - 定义域主机（http 的默认主机是 www）
  • domain - 定义因特网域名，比如 runoob.com
  • :port - 定义主机上的端口号（http 的默认端口号是 80）
  • path - 定义服务器上的路径（如果省略，则文档必须位于网站的根目录中）。
  • filename - 定义文档/资源的名称

CSS

CSS简介

  • CSS是指层叠样式表（Cascading Style Sheets）
  • 样式定义如何显示HTML元素
  • 样式通常储存在样式表中
  • 使用外部样式表可以极大提高工作效率
  • 外部样式表通常储存在CSS文件中
  • 多个样式表可以层叠为一个

CSS语法

  • 选择器指向要改变样式的HTML元素
  • 使用花括号{}把声明括起来
  • 每条声明由一个属性和一个值组成，之间用冒号:分开，声明以分号;为结尾
  • CSS注释以/*开始，以*/结束

id和class选择器

  • id选择器：使用#来选择特定id的元素，id值互异，相同id值的元素不能在HTML文件中共存
  • class选择器：使用.来选择特定class的元素，用来描述一组元素

CSS背景

背景颜色

background-color:属性定义了元素的背景颜色

颜色值有以下三种方式定义：

  • 十六进制："#ff0000"

  • RGB:RGB(255,0,0)

  • 颜色单词:"red"

背景图像

background-image:url('#')属性描述了元素的背景图像

默认情况，背景图像将进行平铺重复显示，以覆盖整个元素实体

  • background-repeat:no-repeat：可以设置图像不平铺

  • background-postion:属性描述了图像的位置

嵌套选择器

  • p{ }：为所有p元素指定一个样式

  • ,marked{ }：为所有**class="marked"**的元素指定一个样式

  • .marked p{ }：为所有class="marked"元素内的p元素指定一个样式

  • p.marked{ }：为所有class="marked"的p元素指定一个样式

CSS Display（显示）

隐藏元素

  • display:none

可以隐藏某个元素，且隐藏的元素不会占用任何空间（即原本占用的空间也会从页面布局中消失）

  • visibility:hidden

可以隐藏某个元素，但隐藏的元素仍需占用与之前一样的空间（即被隐藏了但仍然会影响布局）

块和内联元素

  • 使用display:inline可以将元素改变为内联元素

  • 使用display:block可以将元素改变为块元素

CSS Position（位置）

position属性指定了元素的定位类型，共有五个值

  • static：默认值，即没有定位，不会受到top,bottom,left,right等的影响

  • relatibe：相对定位元素的定位是相对其正常位置（移动相对定位元素，但它原本所占的空间不会改变）

  • fixed：相对浏览器窗口是固定位置，即使窗口是滚动的它也不会移动

  • absolute：绝对定位的元素的位置相对于最近的已定位父元素，若无已定位的父元素，则位置相对于<html>

  • sticky：粘性定位，即基于用户的滚动位置来定位，在position:relative和position:fixed之间相互切换

重叠的元素

z-index属性指定了一个元素的堆叠顺序，高堆叠顺序的元素总是在较低的堆叠顺序元素的前面（若定位重叠且没有指定堆叠顺序，最后定位在HTML代码中的元素将被显示在最前面）

CSS Overflow

Overflow属性可以控制内容溢出元素框时在对应的元素区间内添加滚动条

值 描述
visible 默认值，内容不会修剪，会呈现在元素框之外
hidden 内容会被修剪，并且其余内容是不可见的
scroll 内容会被修剪，但是浏览器会显示滚动条以便查看其余的内容
auto 如果内容被修剪，则浏览器会显示滚动条以便查看其余的内容
inherit 规定应该从父元素继承overflow属性的值

overflow属性只工作于指定高度的块元素上

CSS Float

CSS的Float（浮动）会使元素向左或向右移动（水平），其周围的元素也会重新排列

一个浮动元素会尽量向左或向右移动，直到它的外边缘碰到包含框或另一个浮动框的边框为止

浮动元素之后的元素将围绕它

浮动元素之前的元素将不会受到影响

JavaScript变量

使用var来定义变量（auto,具有动态类型），同时使用等号赋值 对于声明后但未赋值的变量的值为undefined 对于重新声明的变量，其值不会丢失

对于定义在函数外的变量为全局变量 对于定义在函数内的变量为局部变量 使用let关键字定义变量，将此变量的周期限制在{}内

使用var关键字声明的全局作用域变量属于window对象 使用let关键字声明的全局作用域变量不属于window对象

var变量可以先使用后声明，let变量必须要先声明再使用

const用来定义常量，必须初始化且后续无法更改

重置变量
  1. 使用var关键字声明的变量在任何地方都可以修改
var x = 2;
 
// x 为 2
 
var x = 3;
 
// 现在 x 为 3
  1. 在相同的作用域或块级作用域中，不能使用let关键字来重置var关键字声明的变量
var x = 2;       // 合法
let x = 3;       // 不合法
{
    var x = 4;   // 合法
    let x = 5   // 不合法
}
  1. 在相同的作用域或块级作用域中，不能使用let关键字来重置let关键字声明的变量
let x = 2;       // 合法
let x = 3;       // 不合法

{
    let x = 4;   // 合法
    let x = 5;   // 不合法
}
  1. 在相同的作用域或块级作用域中，不能使用var关键字来重置let关键字声明的变量
let x = 2;       // 合法
var x = 3;       // 不合法

{
    let x = 4;   // 合法
    var x = 5;   // 不合法
}
  1. let关键字在不同作用域，或不同块级作用域中是可以重新声明赋值的
let x = 2;       // 合法

{
    let x = 3;   // 合法
}

{
    let x = 4;   // 合法
}
